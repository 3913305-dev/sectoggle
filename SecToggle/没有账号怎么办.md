# 没有 Mac、没有 GitHub —— 怎么办？

## 方案 1：注册 GitHub（推荐，约 2 分钟）

不需要 Mac，免费，专门用来云编译一次：

1. 打开 https://github.com/signup
2. 填邮箱 → 设密码 → 验证邮箱（收验证码）
3. 新建仓库 → 上传 `SecToggle-upload.zip`（在 ios 目录下）
4. Actions → Run workflow → 下载 `SecToggle.dylib`

详细步骤见 `无Mac编译教程.md`

---

## 方案 2：等 Windows 自动编译（进行中）

当前电脑网络较慢，正在后台下载编译器（约 75MB，需 1～2 小时）。

下载完成后会自动编出：
`C:\Users\Administrator\Desktop\ios\SecToggle\SecToggle.dylib`

编好后直接 AirDrop / 微信传到手机，TrollFools 注入即可。

---

## 方案 3：找有 Mac 的人（1 条命令）

把 `SecToggle` 文件夹发给对方，让对方在终端执行：

```bash
cd SecToggle
xcrun -sdk iphoneos clang -arch arm64 -dynamiclib -fobjc-arc \
  -framework UIKit -framework Foundation -framework CoreLocation \
  -install_name @executable_path/SecToggle.dylib \
  -o SecToggle.dylib SecToggle.m -miphoneos-version-min=13.0
```

把生成的 `SecToggle.dylib` 发回给你。

---

## 拿到 dylib 之后

1. 传到 iPhone
2. TrollFools → 中邮司机帮 → 注入
3. 杀进程重开 → 左上角悬浮窗
